# Hyprland Dotfiles

These are my personal **Hyprland dotfiles**.  
Feel free to use them on your system.

---

## Setup

### 1. Clone the repository
```bash
git clone github.com/aprameyansr/Hyprland-Dotfiles
```

### 2. Backup your current configuration
It is recommended to create a backup of your existing config files before proceeding.
```bash
cp -r ~/.config ~/.config.backup
```
### 3. Copy the configuration files
Copy the folders from this repository into your ~/.config directory.
```bash
cp -r <repository-folder>/* ~/.config/
```

### 4. Restart Hyprland
Log out and log back in, or reload Hyprland to apply the changes.
